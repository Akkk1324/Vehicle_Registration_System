# Vehicle Registration System

This is a web-based system developed using Java, Spring Boot, Hibernate ORM, and SQLite. It allows users to register vehicles, search for existing vehicle registrations, and manage vehicle data efficiently.

## Features

- **Register Vehicle**: Users can input vehicle details such as VIN, make, model, and owner information.
- **Search Vehicle**: Users can search for vehicles by VIN or registration number.
- **Database Integration**: Data is stored in an SQLite database, accessed and managed using Hibernate ORM.

## Technologies Used

- **Backend**: Java, Spring Boot, Hibernate ORM
- **Database**: SQLite3
- **Frontend**: Thymeleaf, HTML, CSS, JavaScript
- **Tools**: Maven (for dependency management)
